## Create a new project
1) Creat a folder with the videos you're going to manually label
2) Create a project in DLC. Copy the videos (otherwise it buggs).
3) Edit the config file.  Edit numframes2pick, bodyparts, plotting config, pcutoff, (dotsize)
2) Extract frames
3) Label frames
4) Create training index
5) Train

 -- to restart trainig from a checkpoint, edit the pose_cfg.yaml in the train subdirectory.
modify 'init_weight': give the full path and the snapshot, without the file extension

## Load existing project
1) Load config file
2) Analyze videos
- save as .csv
- filter the predictions
- ResNet: shuffle 0 -> Sep 2022 = 1
- training set index: 0
3) Create Videos (optinal)
